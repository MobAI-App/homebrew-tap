# mobai-app/homebrew-tap

A [Homebrew tap](https://docs.brew.sh/Taps) for [MobAI](https://mobai.run) tools.

## Install

```bash
brew install mobai-app/tap/deskagent
```

Formulas live under [`Formula/`](./Formula/). Source code for each tool is in
its own repo under [`github.com/mobai-app`](https://github.com/mobai-app); this
tap only ships pre-built release tarballs.
